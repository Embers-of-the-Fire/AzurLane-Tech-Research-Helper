## 关于 V3

V3 相对于 V1 采用了完全不同的架构方式，从而二者间不存在兼容的可能。在[Crates.io](crates.io)和[npm](npm.js)上的包我将维持 V1 的版本，并且会跟进碧蓝的改动，但是不会进行算法的更改。

V3 同时也不会提供诸如此类的接口，请使用 tauri app 或者自行从源文件更改使用。
